DARK • MOODY • GEOMETRIC

A — PS1 Raw

Faithful to actual PlayStation 1 rendering: flat/Gouraud shading, visible polygon edges, vertex-level color quantization, affine texture warping. Dark, moody, low-fi. Think Silent Hill, Resident Evil menus.

Post-processing: minimal bloom, palette quantization stays but shifts to muted tones. Dithering becomes ordered (bayer) instead of blue-noise.

CLEAN EDGES

CLEAN • DIGITAL • PRECISE

B — LCD Digital Clean

Early-2000s digital display aesthetic: clean pixel grid, smooth color bands, subtle scanlines or LCD subpixel effect. Think PSP menus, early iPod UI, DVD player displays. Precision over grit.

Post-processing: pixelation stays, replace blue-noise dither with clean banding or optional scanlines. Bloom reduced to subtle glow. Palette quantization with hard edges between colors.

LOW-FI GEOMETRY • HI-FI UI

C — PS1 Geometry + LCD UI (Hybrid)

Keep the low-poly models as-is (they already read as PS1), but clean up the rendering pipeline: remove glitch effects, reduce noise, add clean digital overlays (your screen tags). The models say "PS1", the presentation says "early-2000s digital."

Post-processing: keep pixelation + palette quantization, drop or heavily reduce dithering, drop glitch gating from particles/stars, subtle bloom. Tags rendered as crisp LCD-style overlays.
